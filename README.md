## 1. 소개
가상환경을 만들고 Flask를 이용해 이름과 전화번호와 생일을 입력받아 addbook.txt 파일에 CSV 형식으로 저장하는 간단한 웹 애플리케이션 코드입니다. HTML 템플릿도 함께 포함되어 있습니다. 이를 통해 웹 앱의 기반을 다진것을 바탕으로 추후에 여러가지 기능들을 추가한 웹 앱으로의 발전을 기대하고 있습니다.
